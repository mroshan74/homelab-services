Proxmox

1. Download latest iso from proxmox
2. Download rufus
3. Burn iso to USB using rufus
4. Enable virtualization in BIOS -> [MSI-BIOS -> OC -> advanced -> SE] 
5. Install proxmox

Post-install -> https://tteck.github.io/Proxmox/
